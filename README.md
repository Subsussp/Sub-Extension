# How to use
1. Download the latest `.crx` file from [Releases](../../releases)
2. Open Chrome and go to `chrome://extensions/`
3. Enable **Developer mode** (top right toggle)
4. Drag and drop the `.crx` file onto the page

## The Story behind it
I adore watching TV shows, and while I was looking for British TV shows, I came across a series called Time.

The thing is, they were talking with an Irish accent, and the website I was watching on had no subtitles for the show.

What bothered me is that there were a ton of subtitle files for the show online, but I just couldn't upload them to the website and watch normally.

So my extension lets you do that.

## Features
* .srt files support
* Subtitles Styling Full Control
* Youtube support and mostly any website with a `<video>` element 
* It works on most websites, but I don’t guarantee that it will work on all streaming websites.
* Fully works on **android phones** via `kiwi browser` but only on this version [Download](https://kiwi-browser.en.uptodown.com/android/download/1045295267)

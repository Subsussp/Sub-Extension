# How to use  (Chrome / Brave)
1. Download the latest `.zip` file from [Releases](../../releases)
2. Extract the `.zip` file
3. Open Chrome and go to `chrome://extensions/` or `brave://extensions`
4. Enable **Developer mode** (top right toggle)
5. Click **'Load unpacked'** and **select the extracted folder** containing the `manifest.json` file.
6. **Don't forget to reload the page if you don't see anything happening** and
That's it go to your favourite streaming website and test it out :)

## The Story behind it
I adore watching TV shows, and while I was looking for British TV shows, I came across a series called Time.

The thing is, they were talking with an Irish accent, and the website I was watching on had no subtitles for the show.

What bothered me is that there were a ton of subtitle files for the show online, but I just couldn't upload them to the website and watch normally.

So my extension lets you do that :).

## Features

* .srt files support
* **Youtube support** and mostly any website with a `<video>` element 
* Subtitles Styling Full Control
* It works on most websites, but I don’t guarantee that it will work on all streaming websites.
* Fully works on **android phones** via `kiwi browser` but only on this version [Download](https://kiwi-browser.en.uptodown.com/android/download/1045295267)

## Use Cases

- When the streaming website doesn't provide subtitles in your language—or the language you are trying to learn—but you can find the subtitle file online.
- When the streaming website you're using doesn't offer any subtitles for the movie or series you're watching.
- When you want to customize subtitles for YouTube videos or just in general.
- When the available subtitles are poorly timed and you want to adjust them manually.
